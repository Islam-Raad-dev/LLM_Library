##عن المشروع

في ظل التطور المتسارع لنماذج الذكاء الاصطناعي، أصبح من الصعب على المستخدم معرفة الفرق بين النماذج المتاحة واختيار الأنسب له. **مكتبة نماذج الذكاء الاصطناعي** هو موقع ويب يحل هذه المشكلة بعرض **18 نموذجاً** من أبرز الشركات العالمية في واجهة بسيطة وسريعة.

---

## المميزات

-  **بحث فوري** — ابحث بالاسم أو الوصف أو التخصص أو الكلمات المفتاحية
-  **فلترة مزدوجة** — فلتر حسب الشركة (Claude, Gemini...) وحسب السعر في نفس الوقت
-  **وضع داكن / فاتح** — تبديل المظهر بضغطة زر واحدة
-  **تصميم متجاوب** — يعمل على الجوال والحاسوب
-  **بدون مكتبات خارجية** — HTML + CSS + JavaScript خالص
